Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements nested inside other elements are child elements
  • Elements that contain other elements are parent elements
  • Every opening element has corresponding closing element

CSS

  • A margin indicates how much space we want around the outside an element
  • A padding indicates how much space we want around the content inside an element
  • The most common way to style a webpage is by using an external CSS style sheet
  • Determines the style of the webpage in order to make it visually appealing
  • Can make a website simpler to use with polished design and cosistent organization

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates new branch and switches to it
  • git checkout main: switches to main
  • git pull: retrieves commits from repository
  • git push branch-name: transmits commits to repository
